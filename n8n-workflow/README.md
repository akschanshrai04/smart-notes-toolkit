# 🧠 Smart Notes Workflow (n8n)

This n8n workflow automates the generation of:

- 📄 **Summary** from a given PDF
- 🧾 **Flashcards** for quick revision
- 🧠 **Mindmap** in a tree-structured JSON format (usable with Syncfusion or React Flow)
